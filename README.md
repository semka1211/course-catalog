Fullstack веб-приложение для управления каталогом образовательных курсов и преподавателей.
Разработано в рамках учебной и производственной практики по специальности 09.02.07 
«Информационные системы и программирование».

**Вариант Б — Fullstack Next.js** (App Router)

- **Frontend & Backend:** Next.js 16 (App Router)
- **Язык:** TypeScript
- **Стилизация:** Tailwind CSS
- **Валидация:** Zod
- **Хранилище:** In-memory (массивы в памяти процесса)
- **Контроль версий:** Git + GitHub
